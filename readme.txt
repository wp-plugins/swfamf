== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `swf_amf` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Download the clientside api through the plugin panel, import it in your flash theme and start coding
